# Stock Search & Portfolio Management Platform

Full-stack web platform for stock search, watchlists, and portfolio management with real-time and historical market data.

## Features

- Real-time quotes and company data via the Finnhub API; historical price charts via Polygon.io, rendered with Highcharts
- Stock search with autocomplete suggestions, watchlist, and buy/sell portfolio simulation with profit/loss tracking
- Node.js/Express backend with response caching for API efficiency
- Deployed on Google Cloud Platform

## Stack

Angular · TypeScript · Node.js · Express · Highcharts · Finnhub & Polygon.io APIs · GCP
